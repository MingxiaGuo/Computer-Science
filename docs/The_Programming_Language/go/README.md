语言层面解决软件工程问题的设计哲学： 

* 不得包含在源代码文件中没有用到的包，否则Go编译器会报编译错误。
* 强制左花括号{的放置位置
* 函数名的大小写规则，

与C语言相比，Go语言摒弃了语句必须以分号作 为语句结束标记的习惯。
