## Bus Types

* 片内总线：芯片内部的总线，如CPU芯片内部，寄存器与寄存器之间，寄存器与与算逻单元ALU之间都由片内总线连接。
* 系统总线：系统总线是指CPU，主存、I/O设备各大部件之间的信息传输线。这些部件通常安在主板或各个插板上，又称板级总线或板间总线。
  * 数据总线DBUS
  * 地址总线ABUS
  * 控制总线CBUS
* 通信总线
