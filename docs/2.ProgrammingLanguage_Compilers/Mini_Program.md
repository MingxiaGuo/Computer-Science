# 跨端技术开发框架

* 前端：网站/App/小程序的前端
* 后端：服务器

web跨端

* 原生开发：性能好(页面存在本地，服务器压力小，原生app页面渲染能力强)
* uni-app：转场动画性能差，web端开发限制多；基于vue
* Taro：京东，基于react
* flutter 性能好, 新语言dart

小程序官方更新迭代快，用框架坑很多，且开发体验差



框架：

1. 一次开发适配多个平台
2. 引入主流前端语言开发方式如vue或react
